# 更新日志

所有重要的更改都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-14

### 新增
- 🎵 Electron + Vue 3 桌面音乐播放器
- 🎧 音乐播放与控制功能（播放/暂停/上一首/下一首）
- 📝 歌单管理功能（创建/编辑/删除/收藏）
- 🕒 播放历史记录功能
- 🔍 音乐发现与分类浏览
- 🎨 现代化 UI 界面（Element Plus + UnoCSS）
- 🌍 多语言支持（中文简体/繁体/英文）
- 🌙 深色/浅色主题切换
- 📱 响应式布局设计

### 后端功能
- 🚀 Express 5 + MySQL 后端 API 服务
- 📊 完整的 RESTful API 接口
- 🔐 数据库连接池管理
- 📝 请求日志记录
- ⚠️ 错误处理中间件
- 📚 API 文档自动生成

### 技术特性
- ⚡ Vite 构建工具 + electron-vite
- 🔧 ESLint + Prettier 代码规范
- 📦 一键启动脚本（start.bat）
- 🗄️ 数据库初始化脚本
- 📖 详细的项目文档

### API 接口
- `/api/health` - 健康检查
- `/api/music` - 音乐管理
- `/api/sheet` - 歌单管理
- `/api/songs` - 歌曲管理
- `/api/history` - 播放历史
- `/api/banner` - 轮播图管理
- `/api/playlists` - 播放列表管理

### 页面功能
- 精选页面 - 推荐内容展示
- 发现页面 - 分类浏览音乐
- 所有歌曲 - 歌曲列表浏览
- 随机歌曲 - 随机推荐歌曲
- 歌单详情 - 歌单歌曲管理
- 播放列表 - 播放队列管理
- 最近播放 - 播放历史查看
- 收藏音乐 - 收藏的音乐
- 设置页面 - 应用配置

### 开发工具
- 环境变量配置支持

## 版本说明

### 版本号规则
- **主版本号 (MAJOR)**: 不兼容的 API 修改
- **次版本号 (MINOR)**: 向下兼容的功能性新增
- **修订号 (PATCH)**: 向下兼容的问题修正

### 更新类型
- **新增 (Added)**: 新功能
- **变更 (Changed)**: 对现有功能的变更
- **弃用 (Deprecated)**: 即将移除的功能
- **移除 (Removed)**: 已移除的功能
- **修复 (Fixed)**: 任何 Bug 修复
- **安全 (Security)**: 安全相关的更改
